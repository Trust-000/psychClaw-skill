## ✨ 项目特性

- 🗣️ **老友锐评**：像认识十年的哥们/姐们在回你消息，温和但犀利，杜绝 AI 腔。
- 🛡️ **安全哨兵**：内置极端情绪监测（如“退出世界”），发现信号立即切换严肃干预模式。
- 🍃 **自然收尾**：引入“人称变换”技术，用有力量的中性化短句+微动作自然切断对话循环。
- 🧠 **逻辑平移**：基于 600+ 真实语料库。哪怕你的标题再细碎，也能精准捕捉其底层逻辑。
