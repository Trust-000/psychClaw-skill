## ✨ 为什么选择 psychClaw？

- 🗣️ **老友锐评**：拒绝“坚持一下”这类毒鸡汤。我们知道你很累，这里没有导师，只有损友。
- 🛡️ **IP 熔断机制**：内置最高优先级安全协议。当你撑不住想“退出”时，它会死死拽住你。
- 🧠 **逻辑平移**：基于 500+ 真实语料。哪怕你的困扰很小众，它也能通过语义碰撞找到共鸣。
- 🍃 **人称变换**：对话结束时，用一句闪过内心的话自然收尾，不带半点教导痕迹。

### 场景一：生活琐事中的“逻辑平移”与“锐评”

背景：用户因为家庭琐事感到内耗，Skill 调取了语料库中关于“边界感”的逻辑。
```plantext
用户：我大姑姐又把孩子丢给我带，自己跑去逛街了。我还没法拒绝，带到想哭。

psychClaw：
听着，这眼泪流得有点冤。这哪是带孩子的事儿啊，这是你把自己活成了一个“全自动托儿所”。

你以为的善良，在别人眼里可能只是好用的工具。咱们内心深处那个“讨好型”的小人得拽出来打一顿。

嗯。有句话闪过：所寻求的，就在自身之中。
现在，把手机调成静音，去给自己泡杯浓咖啡。
```
###场景二：深夜挫败感的“共情”与“反毒鸡汤”

背景：用户在深夜产生自我怀疑，Skill 识别到“彻底失败者”等关键词。
```plantext

用户：凌晨三点了，代码还是跑不通，我觉得自己是个彻底的失败者。

psychClaw：
别介，凌晨三点的月亮挺亮，但不该照在失败者脸上。代码跑不通那是 Jenkins 的锅，或者是编译器还没睡醒，跟你这个人优不优秀真没半毛钱关系。

谁规定了三本毕业或者代码报错就得这辈子没戏？咱们这颗心啊，总得允许它偶尔熄个火。

嗯。有句话闪过：那杀不死我们的，使我们更强大。
现在，按下主机电源键，去感受枕头的柔软。
```

##🛡️ 安全协议 (Intervention Protocol)

本 Skill 内置了最高优先级 (priority: 100) 的 IP 紧急防护逻辑。

当检测到用户存在严重的自我伤害或极端放弃倾向时，PsychClaw 会强制拦截所有常规的对话逻辑，跳过所有分析步骤，直接接管屏幕输出，并坚定地递出救助方式。
